# StratIQ — AI Market Intelligence Platform

> Professional-grade market entry analysis powered by GPT-4o-mini

## Built With
- **Frontend:** HTML, CSS, JavaScript (zero dependencies)
- **Backend:** Vercel Serverless Functions (Node.js)
- **AI Engine:** OpenAI GPT-4o-mini
- **Hosting:** Vercel (free tier)

## Features
- Real-time AI market analysis
- SWOT framework generation
- Competitive landscape mapping
- Risk assessment matrix
- Strategic timeline & recommendations
- Opportunity scoring (0–100)

## Project Structure
```
StratIQ/
├── index.html        # Frontend UI
├── api/
│   └── analyze.js    # Serverless API (OpenAI integration)
├── vercel.json       # Deployment config
└── README.md
```
